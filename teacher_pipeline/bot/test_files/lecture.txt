Полная лекция: Информация и информатика

1. Предмет информатики и понятие информации
Информатика является фундаментальной дисциплиной, изучающей свойства информации, а также способы её хранения и обработки в цифровой среде. Информация — это комплексная категория, смысл которой зависит от контекста восприятия.

Свойства информации:
- Полезность: ценность сведений для решения задач.
- Актуальность: соответствие текущему моменту времени.
- Достоверность: отсутствие искажений и ошибок.
- Объективность: независимость от чьего-либо мнения.
- Полнота: достаточность данных для принятия решения.
- Понятность: выражение данных на доступном языке.

Человек воспринимает аналоговые сигналы, но компьютеры работают с цифровой информацией. Современные системы включают инструменты распознавания и перевода аналоговых данных в цифровой формат.

2. Теория и измерение информации
Минимальной единицей цифровых данных является бит. Использование двоичного разряда (0 и 1) упрощает конструкцию вычислительных устройств.

Единицы измерения:
1 Байт = 8 бит
1 Кбайт = 1024 байт
1 Мбайт = 1024 Кбайт
1 Гбайт = 1024 Мбайт
1 Тбайт = 1024 Гбайт

Формула Хартли: N = log2(K) позволяет определить количество бит для описания K состояний. Для систем с разной вероятностью событий (pi) используется формула Шеннона для расчета энтропии (H): H = -Σ(pi * log2(pi)).

3. Кодирование и стандарты
Кодирование — это преобразование данных в цифровой формат.
- ASCII: стандарт для английского алфавита и цифр. Расширенная версия (1 байт) включает кириллицу.
- Unicode: универсальный стандарт (2 байта), кодирующий символы всех алфавитов мира.

4. Помехоустойчивое кодирование
Для защиты от ошибок при передаче данных применяются:
1. Бит четности: добавление одного бита для проверки четности суммы единиц.
2. Троирование: трехкратное повторение каждого бита.
3. Расстояние Хэмминга: метод определения отличий в битах. Для исправления n ошибок расстояние d должно быть d >= 2n + 1.
