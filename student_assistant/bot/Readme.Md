# 👾 Бот ИИ-ассистент для студентов РТУ МИРЭА

## ⚡Запуск бота
1. *Клонировать/запулить репозиторий с гитхаба*  
```git clone https://github.com/sing-ippo/sing-project.git```
2. *Создать виртуальное окружение и запустить его*  
```py -m venv venv```  
```env\Scripts\activate```
3. *Перейти в папку с ботом*
```cd sing-project/student_assistant```
4. *Загрузить все нужные библиотеки из файла requirements.txt*  
```py -m pip install -r bot/requirements.txt```
5. *Запустить бота*  
```py -m bot.bot```
