# 👾 Бот ИИ-ассистент для студентов РТУ МИРЭА

## 📦 Установка зависимостей
1. *Клонировать/запулить репозиторий с гитхаба*  
```git clone https://github.com/sing-ippo/sing-project.git```
2. *Создать виртуальное окружение и запустить его*  
```py -m venv venv```  
```env\Scripts\activate```
3. *Перейти в папку с ботом*
```cd sing-project/student_assistant```
4. *Загрузить все нужные библиотеки из файла requirements.txt*  
```py -m pip install -r bot/requirements.txt```
5. *Установка ffmpeg*
- Windows:
    - Перейти на гитхаб по ссылке ```https://github.com/BtbN/FFmpeg-Builds/releases``` и скачать версию для Windows
    - распаковать zip архив в папку, куда вы хотите сохранить исходники библиотеки, например, *C:\ffmpeg*
    - в поиске Windows найти *Изменение системных переменных сред*, перейти во вкладку дополнительно и нажать на кнопку *Переменные среды*
    - В списке *Системные переменные* найти имя Path со значением *C:\WINDOWS\system32...*
    - Кликнуть на неё и нажать изменить
    - Нажать кнопку созать и скопировать путь до папки bin у библиотеки ffmpeg (например, для пути из пункта 2, путь будет выглядеть как *C:\ffmpeg\bin*)
    - Нажать кнопку *ОК* и закрыть все вкладки, связанные с переменнами среды
    - Открыть консоль от имени админстратора и написать команду ```ffmpeg```
- MacOs: ```brew install ffmpeg```

## ⚡Запуск бота/бэкэнда/тестов для него
* *Запустить бота*  
    - ```py -m bot.bot```
* *Запустить бэкэнд*
    - ```cd bot```
    - ```uvicorn bot/voice_server:app --reload```
* *Запуск тестов для бэкэнда*
    - ```cd bot```
    - ```pytest -s test_voice.py --disable-pytest-warnings```
