## Что нужно сделать

### 1. pymorphy2 в поиске

Текущая проблема: `search_engine.py` ищет точное совпадение слов. Вопрос «где находится деканат?» не найдёт запись с ключевым словом «деканата».

Решение — нормализовать слова перед поиском:

```python
import pymorphy2

morph = pymorphy2.MorphAnalyzer()

def normalize(word: str) -> str:
    return morph.parse(word)[0].normal_form

def normalize_query(query: str) -> list[str]:
    return [normalize(w) for w in query.lower().split()]
```

**Что изменить в `build_faq.py` / `search_engine.py`:**
- При сборке faq.json — нормализовать ключевые слова
- При поиске — нормализовать запрос перед сравнением

**Витя** делает эту часть.
