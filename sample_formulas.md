# Сборник формул для проверки

Теорема Пифагора связывает катеты и гипотенузу прямоугольного треугольника:
$$a^2 + b^2 = c^2$$

Корни квадратного уравнения находятся по формуле:
$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

Тождество Эйлера — одно из красивейших в математике:
$$e^{i\pi} + 1 = 0$$

Производная произведения двух функций:
$$(uv)' = u'v + uv'$$

Определённый интеграл функции на отрезке:
$$\int_{a}^{b} f(x)\,dx$$

Связь массы и энергии (Эйнштейн):
$$E = mc^2$$

Сумма первых n натуральных чисел:
$$\sum_{k=1}^{n} k = \frac{n(n+1)}{2}$$

Первый замечательный предел:
$$\lim_{x \to 0} \frac{\sin x}{x} = 1$$

Второй закон Ньютона:
$$F = ma$$

Волновое уравнение в физике:
$$\frac{\partial^2 u}{\partial t^2} = c^2 \nabla^2 u$$
