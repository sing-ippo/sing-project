# СИНГ ИиППО — Проект AI-ассистентов РТУ МИРЭА

Учебный проект студенческой исследовательской научной группы кафедры ИиППО.

Два продукта: **ассистент для студентов** (FAQ-бот) и **ассистент для преподавателей** (документ → квиз).

## Структура репозитория

```
sing-project/
│
├── student_assistant/          # Команда «Ассистент студента»
│   ├── bot/                    # Telegram FAQ-бот (Артём + Лев)
│   └── data/                   # Пайплайн сборки базы знаний (Витя + Саша)
│
├── teacher_pipeline/           # Команда «Ассистент преподавателя»
│   ├── pipeline/               # Цепочка chunking → quiz (Пётр + Кирилл)
│   └── bot/                    # Telegram-бот преподавателя (Павел + Спартак)
│
├── analytics/                  # Дашборд аналитики запросов (Макар)
├── formulas/                   # API извлечения формул (Андрей)
│
└── docs/                       # Общая документация
    ├── FORMATS.md              # Единые форматы данных (8 форматов)
    ├── GIT_GUIDE.md            # Инструкция по git для участников
    └── SPRINT3.md              # Задания Sprint 3
```

## Как работать с репозиторием

Подробная инструкция: [`docs/GIT_GUIDE.md`](docs/GIT_GUIDE.md)

Коротко:
1. Клонируй: `git clone <url>`
2. Создай ветку: `git checkout -b team-имя-имя`
3. Работай в **своей папке**
4. Коммить и пуш: `git add . && git commit -m "..." && git push`
5. Создай Pull Request на GitHub
