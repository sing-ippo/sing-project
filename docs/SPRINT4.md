# Sprint 4 — Голосовой стенд «День открытых дверей»

**Срок:** 1 неделя | **Формат:** работа в парах/соло | **Сдача:** Pull Request

## Цель

Собрать голосовой киоск для Дня открытых дверей: абитуриент подходит, задаёт вопрос голосом — стенд отвечает голосом и показывает ответ на экране.

## Архитектура стенда

```
Браузер (Пётр + Кирилл)
  → кнопка «Говори» → запись аудио (MediaRecorder API)
  → POST /ask → FastAPI бэкенд (Артём + Лев)
      → Whisper STT → текст вопроса
      → search_engine (Витя + Саша: pymorphy2) → ответ из faq.json
      → Silero TTS → аудио ответа
  ← { question, answer, audio_base64 }
  → показать текст + проиграть аудио
```
